Desenvolvimento de Artes Visuais

Para uma melhor jogabilidade e interesse de todos os jogadores, o design do jogo foi feito exclusivamentepelos criadores. O design foi pensado em Pixels Arts, o qual consiste em construir arte apartir dos pixels de uma imagem, aplicando cores que combinam entre si, para que assim possa dar inicio a uma arte visual. Todos os personagens e cenários tiveram em mente pessar para o jogador uma época mais antiga, para que o jogador aprenda mais sobre grandes ganhadores do Nobel Prize e Turing Awards.

Desenvolvimento do Site para divulgação do jogo

O site foi desenvolvido a partir de Ferramentas HTML, CSS, JavaScript e Frameworks como BootsTrap, para facilitar o desenvolvimento e melhor interação com o usuário, para que ele possa usufruir de todas as coisas que o site pode oferecer para que o jogo possa atingir e levar o conhecimento da Computação e Ciência para o máximo número de pessoas possivel.
